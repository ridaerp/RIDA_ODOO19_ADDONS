Frequently Asked Questions

Which Odoo versions are supported?

This application is fully compatible with Odoo Community and Enterprise editions for versions 17, 18, and 19.

Will the target user be notified when I connect?

No, the target user does not receive a notification. However, the action is securely logged in the internal system logs for audit purposes by system administrators.

Can any user use this feature?

Absolutely not. By default, no one has access. You must explicitly assign users to the "Connect As User" security group for the button to appear in their interface.
